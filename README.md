## Installation

Repo link 
https://github.com/raj0811/AuthenticationSystemCn.git


> ***Install project with npm***       

```bash
  cd AuthenticationSystemCn
  npm install
  inp install bcrypt
  npm install nodemon

```

***Start APP***

```bash 
npm start
```
